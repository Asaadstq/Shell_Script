# Arguments
 
Arguments are values passed to a script when it is executed from the terminal. They allow your script to accept input dynamically instead of hardcoding values.

When running a script:

```bash
./script.sh arg1 arg2 arg3
```
- `arg1` becomes `$1`
- `arg2` becomes `$2`
- `arg3` becomes `$3`

Let's see this example for the argument script
```bash
#!/bin/bash
 
echo "Script name : $0"
echo "Name        : $1"
echo "Age         : $2"
echo "Total args  : $#"
echo "All args    : $@"
```
 
Running
```bash
#  ./greet.sh <name> <age>
./greet.sh Ahmad 24
```
 
Output
```
Script name : ./greet.sh
Name        : Ahmad
Age         : 24
Total args  : 2
All args    : Ahmad 24
```
You can see the table provides an explanation of the output.

| Parameter | Value |
|---|---|
| `$0` | Script name |
| `$1`, `$2` | First and second arguments |
| `$#` | Total number of arguments |
| `$@` | All arguments |


- `$0` → ./greet.sh
- `$1` → Ahmad
- `$2` → 24
- `$#` → 2 (because two arguments were passed)
- `$@` → Ahmad 24
