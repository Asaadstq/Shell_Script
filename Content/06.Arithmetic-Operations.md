# Arithmetic Operations

Bash supports integer arithmetic using:
 
 ```bash
$(( expression ))
```

This is called **arithmetic expansion**. It evaluates the expression and returns the result.

>[!Important]
>Bash arithmetic works with **integers only** (no decimals).

##  Basic Arithmetic
| Operator | Meaning |
|----------|----------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division (integer only) |
| `%` | Modulo (remainder) |
| `**` | Exponentiation |

- Example
```bash
#!/bin/bash
 
a=10
b=3
 
echo "Addition       : $(( a + b ))"
echo "Subtraction    : $(( a - b ))"
echo "Multiplication : $(( a * b ))"
echo "Division       : $(( a / b ))"
echo "Modulo         : $(( a % b ))"
echo "Exponentiation : $(( a ** 2 ))"
```
 
Output
```
Addition       : 13
Subtraction    : 7
Multiplication : 30
Division       : 3
Modulo         : 1
Exponentiation : 100
```
 
 
 ### Integer Division in Bash

Bash does **not** handle floating-point numbers inside `$(( ))`.

Example:

```bash
echo $(( 10 / 3 ))
```

Output:

```
3
```

the decimal part is **truncated** (not rounded).

### Floating-Point Arithmetic using `bc`

To perform decimal calculations, use the `bc` command.

`bc` stands for **Basic Calculator** and supports floating-point arithmetic.

- Example

```bash
result=$(echo "scale=2; 10 / 3" | bc)
echo $result
```

### Explanation

- `scale=2` → number of decimal places  
- `echo "expression" | bc` → sends the math expression to `bc`  
- `$( ... )` → captures the output into a variable  

### Output

```
3.33
```
