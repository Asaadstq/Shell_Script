# Conditions

Conditions allow your script to make decisions.

Bash mainly uses:

- `if / elif / else`
- `case`


## The `if` Statement

The `if` statement runs code only if a condition is true.


- Syntax

```bash
if [[ condition ]]; then
    # code
elif [[ another_condition ]]; then
    # code
else
    # code
fi
```
>[!Note]
>`[[ ]]` is used to test conditions in Bash. </br>
>`fi` closes the `if` block (it is `if` written backward).


- Example
check if a number is positive, negative, or zero

```bash
#!/bin/bash

read -p "Enter a number: " num

if [[ ${num} -gt 0 ]]; then
    echo "Positive"
elif [[ ${num} -lt 0 ]]; then
    echo "Negative"
else
    echo "Zero"
fi
```


## Comparison Operators

### Integer Comparisons

Used when comparing numbers.

| Operator | Meaning |
|----------|----------|
| `-eq` | Equal |
| `-ne` | Not equal |
| `-gt` | Greater than |
| `-lt` | Less than |
| `-ge` | Greater than or equal |
| `-le` | Less than or equal |

- Example

```bash
if [[ $a -gt $b ]]; then
    echo "a is greater than b"
fi
```

### String Comparisons

Used when comparing text.

| Operator | Meaning |
|----------|----------|
| `==` | Equal |
| `!=` | Not equal |
| `-z` | Empty string |
| `-n` | Non-empty string |


- Example: string condition

```bash
#!/bin/bash

read -p "Enter your name: " name

if [[ -z "${name}" ]]; then
    echo "You didn't enter a name."
elif [[ "${name}" == "Admin" ]]; then
    echo "Welcome, administrator."
else
    echo "Hello, ${name}."
fi
```

## The `case` Statement

The `case` statement is cleaner than multiple `if-elif` blocks when checking one variable against many fixed values.


- Syntax

```bash
case variable in
    pattern1)
        # code
        ;;
    pattern2)
        # code
        ;;
    *)
        # default case
        ;;
esac
```

- `;;` ends each case block  
- `*` acts as the default case (like `else`)  
- `esac` closes the case statement  

- Example

```bash
#!/bin/bash

read -p "Choose a day (mon/tue/wed): " day

case "${day}" in
    mon)
        echo "Monday — Start of the week"
        ;;
    tue)
        echo "Tuesday"
        ;;
    wed)
        echo "Wednesday — Midweek"
        ;;
    *)
        echo "Unknown day: ${day}"
        ;;
esac
```


### When to use `if` and `case`?

| Use | When |
|------|------|
| `if` | When checking ranges, numeric conditions, or complex logic |
| `case` | When matching one variable against fixed values |
