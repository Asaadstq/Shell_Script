# Loops

Loops allow you to **repeat a block of code multiple times**.

Bash provides two main types of loops:

- `for` loop used when the number of iterations is known.
- `while` loop used when repeating until a condition becomes false.


## The `for` Loop

The `for` loop is commonly used to:

- Iterate over a list
- Iterate over a range of numbers


Syntax

```bash
for variable in list; do
    # code
done
```


Example 1: Iterating Over a List

```bash
#!/bin/bash

fruits=("apple" "banana" "cherry")

for fruit in "${fruits[@]}"; do
    echo "Fruit: ${fruit}"
done
```

Explanation

- `fruits=(...)`: Creates an array
- `"${fruits[@]}"`: Expands all elements in the array
- `fruit`: Temporary variable that holds each value during the loop

Output

```
Fruit: apple
Fruit: banana
Fruit: cherry
```



Example 2: Iterating Over a Range

```bash
#!/bin/bash

for i in {1..5}; do
    echo "Number: ${i}"
done
```

Explanation

- `{1..5}`: Generates numbers from 1 to 5
- `i`: Stores each number during the loop

Output

```
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5
```


## The `while` Loop

The `while` loop runs **as long as the condition is true**.



Syntax

```bash
while [[ condition ]]; do
    # code
done
```



Example

```bash
#!/bin/bash

count=1

while [[ ${count} -le 5 ]]; do
    echo "Count: ${count}"
    (( count++ ))
done
```

Explanation

- `[[ ${count} -le 5 ]]`: The loop runs while `count` is less than or equal to 5
- `(( count++ ))`: Increments `count` by 1



Output

```
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
```



### When to use each Loop?

| Loop Type | When to Use |
|------------|-------------|
| `for` | When you know how many times to repeat |
| `while` | When repeating depends on a condition |
