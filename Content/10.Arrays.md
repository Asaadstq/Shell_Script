# Arrays

An array allows you to store multiple values in a single variable. Each value is stored at an **index**, starting from 0.
 
## Creating and Accessing


#### Creating an Array
```bash
colors=("red" "green" "blue")
```
- Elements are separated by spaces.
- Each element is inside quotes if it contains text.


#### Accessing Elements

Example
```bash
#!/bin/bash
 
colors=("red" "green" "blue")
 
echo "First  : ${colors[0]}"
echo "Second : ${colors[1]}"
echo "All    : ${colors[@]}"
echo "Count  : ${#colors[@]}"
```

#### Explanation
- `${colors[0]}`	First element (index starts at 0)
- `${colors[@]}`	All elements
- `${#colors[@]}`	Number of elements

Output
```
First  : red
Second : green
All    : red green blue
Count  : 3
```
 
## Looping through an array

The most common way to loop through an array

```bash
#!/bin/bash
 
languages=("Bash" "Python" "JavaScript" "Java")
 
for lang in "${languages[@]}"; do
    echo "- ${lang}"
done
```

Always use `${array[@]}` with quotes this prevents problems if elements contain spaces.
 
Output
```
- Bash
- Python
- JavaScript
- Java
```
  
## Adding and Removing Elements
 
```bash
#!/bin/bash
 
items=("a" "b" "c")
 
# Add an element
items+=("d")
echo "After add    : ${items[@]}"
 
# Remove element at index 1
unset 'items[1]'
items=("${items[@]}")   # re-index
echo "After remove : ${items[@]}"
```

 >[!important]
> Use `+=()` to add elements
> After `unset`, the index is removed but not automatically re-numbered


Output
```
After add    : a b c d
After remove : a c d
```
